PyTorch官方文档之torch.Tensor

torch分别定义了CPU中存储和GPU中存储的两种类型.

torch.Tensor()等于就是torch.FloatTensor() 
import torch
import numpy as np
a = torch.Tensor(1) # torch.Tensor()就是默认数据类型为torch.FloatTensor()的张量
b = torch.FloatTensor(1)
a,b # torch.Tensor()创建一个默认类型为torch.FloatTensor()的张量,torch.FloatTensor()是CPU中类型
(tensor([1.4013e-45]), tensor([1.4013e-45]))

使用Python中的list创建tensor张量,或者使用np.array()也可以

a = torch.tensor([[1., -1.], [1., -1.]]) # 注意,这里创建的是内容为1.和-1.的tensor,不是维度,
b = torch.tensor(((1., -1.), (1., -1.))) # 并且,这里用[]或()都一样
a, b
(tensor([[ 1., -1.],
         [ 1., -1.]]), tensor([[ 1., -1.],
         [ 1., -1.]]))
torch.tensor(np.array([[1, 2, 3], [4, 5, 6]]))
tensor([[1, 2, 3],
        [4, 5, 6]], dtype=torch.int32)

PS: 使用requires_grad_()或者detach()允许更改requires_grad的标志
使用torch.dtype指定类型,使用torch.device指定存储的位置

torch.zeros([2, 4], dtype=torch.int32)
tensor([[0, 0, 0, 0],
        [0, 0, 0, 0]], dtype=torch.int32)
cuda = torch.device('cuda:0')
torch.ones([2, 4], dtype=torch.float64, device=cuda)
tensor([[1., 1., 1., 1.],
        [1., 1., 1., 1.]], device='cuda:0', dtype=torch.float64)

索引:可以使用Python的索引方式

x = torch.tensor([[1, 2, 3], [4, 5, 6]])
print(x)
print(x[1][2])
print(x[1,2])
x[0][1] = 8
print(x)
tensor([[1, 2, 3],
        [4, 5, 6]])
tensor(6)
tensor(6)
tensor([[1, 8, 3],
        [4, 5, 6]])

使用torch.Tensor.item()返回张量的值

x = torch.tensor([[1]])
print(x)
print(x.item())
x = torch.tensor(2.5)
print(x)
print(x.item())
tensor([[1]])
1
tensor(2.5000)
2.5

使用requires_grad=True让tensor张量能够记下操作记录,以致于可以求导

x = torch.tensor([[1., -1.], [1., 1.]], requires_grad=True)
out = x.pow(2).sum() # out = x^2
out.backward() # 求导:out = 2 * x
print(x.grad)
tensor([[ 2., -2.],
        [ 2.,  2.]])

NOTE:当需要改变现存的tensor的device时,使用to()方法

CLASS torch.Tensor

有几种创建tensor的方式

  1. 用上述的torch.tensor()创建
  2. 用torch.*来创建指定size的张量
  3. 用一个张量的size来创建不同类型的张量,使用torch.*_like
  4. 用一个张量的类型创建不同size的张量,使用torch.new_*

例如:torch.rand(),torch.rand_like(),torch.randn(),torch.randn_like()
,torch.randint(),torch.randint_like(),torch.randperm(),torch.empty()

new_tensor(data)

创建一个数据为data其他都为原tensor类型的一个张量

t = torch.ones((2, ), dtype=torch.int8)
data = [[0, 1], [2, 3]]
t.new_tensor(data)
tensor([[0, 1],
        [2, 3]], dtype=torch.int8)

new_full(size, fill_value)

返回一个用fill_value填充的size维度的Tensor,默认返回的dtype和device相同
fill_value标量

t = torch.ones((2, ), dtype=torch.float64)
t.new_full((3, 4), 3.141592)
tensor([[3.1416, 3.1416, 3.1416, 3.1416],
        [3.1416, 3.1416, 3.1416, 3.1416],
        [3.1416, 3.1416, 3.1416, 3.1416]], dtype=torch.float64)

new_empty(size)

返回size维度的用随机初始化数据填充的Tensor

t = torch.ones((1 ))
print(t)
t.new_empty((2, 3))
tensor([1.])





tensor([[0.0000e+00, 0.0000e+00, 1.4013e-45],
        [0.0000e+00, 1.4013e-45, 0.0000e+00]])

new_ones(size)

返回一个size维度但是用1填充的Tensor

t = torch.tensor((2.))#, dtype=torch.int32)
print(t.dtype)
print(t.new_ones((2, 3)))
print(t)
torch.float32
tensor([[1., 1., 1.],
        [1., 1., 1.]])
tensor(2.)

new_zeros(size)

返回一个size大小用0填充的Tensor

tensor = torch.tensor((), dtype=torch.float64)
tensor.new_zeros((2, 3))
tensor([[0., 0., 0.],
        [0., 0., 0.]], dtype=torch.float64)

is_cuda()

如果tensor存储在GPU中返回True,否则返回False

is_quantized()

如果tensor是量化的返回True,否则返回False

device

返回Tensor存在哪

grad

记录求导backward()之后变量的值

ndim

dim()的别名,返回的tensor的维度

T

返回矩阵的转置

x = torch.tensor([[0, 1], [2, 3]])
print(x)
print(x.T)
tensor([[0, 1],
        [2, 3]])
tensor([[0, 2],
        [1, 3]])

abs() / abs_()

与torch.abs()相同,返回矩阵中元素的绝对值

torch.tensor([-1, -2, 3]).abs()
tensor([1, 2, 3])

acos() / acos_()

与torch.acos()相同,返回arccos反余弦值

add(other, *, alpha=1) / add_()

与torch.add()相同,返回张量+alpha*other

x = torch.ones((2, 3))
print(x)
x.add(1, alpha=2)
tensor([[1., 1., 1.],
        [1., 1., 1.]])





tensor([[3., 3., 3.],
        [3., 3., 3.]])
